HTML

  • HTML,Hypertext Markup Language, is used to create the structure of a webpage. It is made up of elements which work together to provide content and the basic framework for all pages on the web.
  • An element is a type of content on a webpage such as: text elements, image elements, header elements, and list elements.
  • HTML elements are composed of an opening and closing tag represented with angle brackets. These tags contain everything we want to include in the element. This can contain content and other nested elements.
  • The head element contains information or metadata, that helps the browser to render the page correctly. Elements included within the head are not visible when the user looks at a webpage.
  • The body element contains the content that is shown to the user. All the elements that we want the user to see and interact with must be placed inside this element.
